# Simon Game 🎮

A simple memory-based game where players repeat an increasing sequence of colors.

## 🌐 Live Demo
https://dharyagoyal.github.io/simon-game/

## 🚀 Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 Features
- Random color sequence generation
- Level-based progression
- Game over detection
- Basic visual feedback

## 🧠 How to Play
1. Press any key to start the game  
2. Observe the sequence carefully  
3. Repeat the sequence by clicking the buttons  
4. The sequence grows every level  
5. A wrong click ends the game
